# StudyPlanner 学习备考助手 — 用户手册

## 1. 简介

StudyPlanner 是一款简洁高效的学习备考管理工具，帮助学生管理课程、任务、考试，并通过番茄钟和学习统计提升学习效率。

## 2. 安装与启动

### 2.1 启动方式
- 双击 `StudyPlanner.exe` 即可启动
- 程序会自动打开浏览器访问 `http://127.0.0.1:5000`
- 关闭命令行窗口即可退出程序

### 2.2 系统要求
- Windows 7 及以上版本
- 不需要安装 Python 或其他依赖
- 首次运行会在 exe 同目录生成 `studyplanner.db` 数据库文件

## 3. 功能模块

### 3.1 用户系统
- **注册**：首次使用需注册账号
- **登录**：输入用户名和密码登录
- **个人中心**：点击左下角头像进入，可修改密码、查看数据概览

### 3.2 首页
- 显示今日任务、即将到来的考试、本周学习统计
- 快速添加任务输入框（按 `/` 键聚焦，`Ctrl+Enter` 提交）
- 根据时间自动显示问候语（早上好/下午好/晚上好）

### 3.3 课程管理
- 添加课程：设置名称和颜色标签
- 支持学期分组管理
- 课程可关联任务、考试、学习记录

### 3.4 任务管理
- **三种视图**：列表、看板、日历
- **快捷操作**：
  - 拖拽排序（列表视图）
  - 标签分类（逗号分隔）
  - 删除撤销（5秒内可撤销）
- **筛选**：按状态（全部/待办/已完成）、课程、标签筛选
- 日历视图支持月份切换，显示每日任务

### 3.5 笔记
- 创建、编辑、删除笔记
- 支持置顶功能
- 关键词搜索

### 3.6 番茄钟
- **预设时间**：15/25/45/60 分钟
- **自定义时间**：输入 1-120 分钟任意时长
- **休息时长**：3/5/10/15 分钟可选
- 可关联任务，完成自动记录学习时长
- 空格键快捷暂停/开始
- 浏览器标签页显示倒计时

### 3.7 考试管理
- 添加考试：名称、日期、地点、备注
- 首页显示倒计时
- 可关联课程

### 3.8 学习统计
- 近 7 天学习时长柱状图
- 课程学习时长分布
- 学习热力图（近一年）
- 任务完成率统计

### 3.9 数据导出
- 导出学习记录（CSV）
- 导出任务列表（CSV）
- 导出番茄钟记录（CSV）
- CSV 文件可用 Excel 直接打开

## 4. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `/` | 聚焦首页快速添加输入框 |
| `Ctrl+Enter` | 提交快速添加任务 |
| `Ctrl+K` | 聚焦搜索框 |
| `Space` | 番茄钟暂停/开始 |

## 5. 数据安全

- 所有数据存储在本地 SQLite 数据库文件中
- 数据库文件位于 exe 同目录下的 `studyplanner.db`
- 建议定期备份该文件
- 用户密码以明文存储（本地应用，无网络传输）
